My input, in rough non-polished words, not necessarily logical order: 
So my project is the one that does the self-trapped exciton investigation using the excited state forces 
method. So the project started with first theory and code development for self-trapped excitons self-energy 
diagrams through the use of tadpole and fan-migdal diagrams. The Tadpole diagram was solved self-consistently. 
So the first method that was used was towards self-consistent methods for self-trapped exciton analysis. 

Even thjgouth the project said investigation of self-trapped exciton suing excited state forces, the method 
encountered some challenges. Specifically, the initial guess influenced the number of steps and convergence 
in the excited state forces calculation. SO in order to minimize the steps, a better initial guesss was needed 
and the self-consistent self-energy method provided a better starting point for this. 

So initially, was happenedf was that calculations were run on Cs2AgInCl6, and one found out that the convergence 
oscillated or was hard to achieve. This suggested to use something else as a starting point. Since these computations
are expensive, the computations are not easy to just repeate with differetn starting points easily. This motivated 
the need for better theory and other code to get better initial guess. This leads to the above paragraph. 

Now, the self-energy method has tadpole and fan-migdal diagrams that required some tehory development and then subsequent 
code developemtn. The method for calculation involved inputs form finite momentum exciton calculations for two systems, 
LiF, SiO2 and Cs2AgInCl6. Specifically, one calculated the finite momentum excitons on these systems for these
systems on 2x2x2, 4x4x4, 6x6x6, and 8x8x8 grids and then extrapolated the results for higher infinite grids. 

The result of these calculations, was to first benchmark the calculations for LiF and SiO2, since these systems have 
STE results in literature. Experimetnal results for SiO2 also exist. This gives a good place to test the code and theory
implementation to see what we can get. The downside fo the theory is that we work under the harmonic approximation for 
lattice distortion. The upside of the theory is that we can also include dynamic effects which include temperature 
dependence that is not captured by the excited-state forces method. The excited-state forces method onthe other 
hand captures the non-harmonic lattice distortion that STEs can cause. 

Then we used the distortion calculated form the partially self-consistent self-energy method to be able to get 
the initial guess for the excited state forces method. This way the number of iterations for the excited-state 
forces for Cs2AgInCl6 can be drastically reduced to a few steps, since we expect to be close to the minimum, and 
ESF in this case captures the static difference due to non-harmonic distortions. 

As for impact on community paradigms, STE temeprature dependence has not been calculated from first oprinciples, so 
this provides a novel way to calculate it, with the associted theory behind the methods. Addidionally, the work 
also compares the results and approximations formt he self-energy mehtod and the excited state forces method. 

For the valuable insights into the long standing challenge, the challenge being to use excited-state forces method. 
A good initial guess is crucial for convergence for STEs, as certain initial distortions do not achieve convergence. 
Toy model calculations show this could be due to barriers in the STE formation. 

The milestones and goalds was to be able to undestand and analyze STEs from first principles and decelop 
associated code for it. This work was able to make theoretical advancements and develop code to be able to 
compare and contrast results. 

The impact is that, this provides the phonon states and the temperature dependence on the width of the STE 
emission and estimatte for the stokes shift based on first pricniples for STEs, with associted thory. 

Some follow up investigations that were motivated initially was from the failture of convergence in the STE
in Cs2AgInCl6. And also toy models with Frohlich coupling show that a better initial guess is needed to be able 
to get convergence in some cases, as STEs can host barriers. The barrier comes form the competition in the 
momentum dependent "kinetic energy" of excitons and the energy reduction in the lattice distortion.
